Simply run ASM2020.exe to see the program usage

From command prompt run ASM2020.EXE with following command line to compile sample Variance calulator program

ASM2020.EXE compile mean variance data main variance.vasm.bin

1st argument: compile - tells ASM2020 to compile input source files into single executable
2nd t0 penultimate arguments: list of input source file names
last argument: output exceutable file name

From command prompt run ASM2020.EXE with following command line to run the precompiled executable file
ASM2020.EXE variance.vasm.bin

type ? to see command help
